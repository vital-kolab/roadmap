# ROADMAP scope notes used for this update

This site was updated from the attached project summary and co-creation deck. Key scope cues incorporated:

- ROADMAP as a unified human-technology platform for translational autism research.
- Integration of mechanistic neuroscience, cross-species experimentation, community partnership, clinical interpretation, AI benchmarking, and private-sector translation.
- Formalizing trait dimensions such as E/I imbalance, sensory noise, feedback circuit differences, hyperactivity/gain, and altered priors/generalization as parameterized ANN modifications.
- Year-1 prototype target: curated datasets, integrated benchmarks, base ANN simulations, a model sandbox, and transparent reporting dashboards.
- Emphasis that this is a research/translation platform, not a diagnostic device.
