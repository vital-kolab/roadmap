Submission center

Submit neural hypotheses, behavior data, or simulation jobs.

This page is a production-style UI mockup. It collects the metadata ROADMAP needs to review neural hypotheses, receive behavior datasets, route jobs to compute, and return benchmarked results to the right audience.

Neural hypothesis submission

Use this when an animal-model study reveals a testable neural hypothesis for autistic behavior, sensory processing, learning, social communication, or intervention response. Neurotypical model simulations can be routed through the Brain-Score collaboration workflow.

Behavior submission

Use this when a researcher wants to contribute a behavioral dataset, task, stimulus set, protocol, or participant-level summary that can become part of the ROADMAP benchmark registry.

Simulation job request

Use this when someone wants ROADMAP to run a model simulation or benchmark job on scalable compute.

Submission preview

For neural hypotheses and behavior submissions, this form sends the active tab to the ROADMAP team by email using Formspree. The JSON preview shows the structured record that can later move into the benchmark registry.
