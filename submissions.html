Submission center

Submit models, datasets, or simulation jobs.

This page is a production-style UI mockup. It collects the metadata ROADMAP needs to validate submissions, route jobs to compute, and return benchmarked results to the right audience.

Model submission

Use this when a modeller wants to register a computational model, code package, or animal-model hypothesis for benchmarking.

Dataset submission

Use this when a researcher wants to contribute a behavioral, imaging, neural, task, stimulus, or protocol dataset.

Simulation job request

Use this when someone wants ROADMAP to run a model simulation or benchmark job on scalable compute.
